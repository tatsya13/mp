# ГТО для всех: Проектный рывок

Интерактивная игра-симулятор управления физкультурно-спортивной организацией.

## 🎮 О проекте

«ГТО для всех: Проектный рывок» — это образовательная игра, в которой вы принимаете на себя роль руководителя проекта и учитесь балансировать между охватом аудитории, качеством мероприятий и доступными ресурсами.

## ✨ Особенности

- **Интерактивный сюжет** — пройдите через 4 ключевых сценария управления проектом
- **Команда проекта** — работайте с виртуальной командой «Мобильный патруль»:
  - Алексей — Руководитель проекта
  - Марина — Врач-методист
  - Денис — Специалист по работе с молодёжью
  - Ольга — Бухгалтер-экономист
- **Система KPI** — отслеживайте три ключевых показателя:
  - 📊 Охват — количество вовлечённых участников
  - ⭐ Качество — процент качества проведения мероприятий
  - 💰 Ресурсы — бюджет проекта (в млн ₽)
- **Визуальная обратная связь** — реакции персонажей на ваши решения
- **Адаптивный дизайн** — работает на десктопах и мобильных устройствах

## 🚀 Запуск

Просто откройте файл `index.html` в любом современном браузере:

```bash
# На Linux/Mac
open index.html

# На Windows
start index.html
```

Или перетащите файл `index.html` в окно браузера.

## 🎯 Как играть

1. Нажмите **«Начать игру»** на титульном экране
2. Познакомьтесь с командой проекта
3. Прочитайте легенду и правила
4. В каждом из 4 раундов выберите один из вариантов ответа
5. Получите обратную связь от команды
6. В конце игры узнайте свой итоговый счёт и оценку руководства
